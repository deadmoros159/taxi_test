# Driver Service

Сервис для управления водителями и их автомобилями.

## Функционал

- Регистрация водителей диспетчером
- Управление автомобилями водителей
- Документы водителей
- Статусы водителей (активен, на заказе, офлайн)

## API Endpoints

### Для диспетчеров:
- `POST /api/v1/drivers/register` - Регистрация водителя с автомобилем
- `GET /api/v1/drivers` - Список всех водителей
- `GET /api/v1/drivers/{driver_id}` - Информация о водителе
- `PATCH /api/v1/drivers/{driver_id}/status` - Изменить статус водителя

### Для водителей:
- `GET /api/v1/drivers/me` - Информация о себе
- `PATCH /api/v1/drivers/me/vehicle` - Обновить информацию об автомобиле

## Интеграция с auth-service

Driver-service использует auth-service для:
- Проверки токенов (через HTTP запросы или shared library)
- Получения информации о пользователе
- Проверки ролей (диспетчер, водитель)
